# exply Privacy Policy

**Last Updated:** January 7, 2025

## Overview

exply is a browser extension that provides AI-powered explanations of highlighted text. This privacy policy explains how we handle your data.

## Data Collection

**exply does NOT collect, store, or transmit any personal data.**

## Local Storage

exply uses Chrome's local storage API to store:
- Your selected UI language preference
- This data is stored **locally on your device only**
- This data is **never transmitted** to any server
- You can clear this data anytime via Chrome's extension settings

## API Usage

When you use exply to explain text:
- The highlighted text and surrounding context are sent to our backend server
- Our backend server processes the request using Google's Gemini API
- **No data is stored or logged** on our servers
- **No personal information is collected**
- Each request is independent - we don't track usage across sessions

## Third-Party Services

exply uses:
- **Google Gemini API**: For generating explanations (via our secure backend)
- **Vercel**: For hosting our backend server (serverless, no persistent storage)
- No other third-party services are used

## Data Sharing

**We do NOT share, sell, or distribute any user data.**

## User Control

- All data is stored locally on your device
- You can clear stored data via Chrome extension settings
- No account or login required
- Extension only works when you explicitly trigger it

## Security

- All API communications use HTTPS encryption
- API keys are stored server-side only (never exposed to the extension)
- No sensitive data is logged or stored
