# Quick Start Guide

## Step 1: Load the Extension in Chrome

1. **Open Chrome** and navigate to: `chrome://extensions/`
2. **Enable Developer Mode**: Toggle the switch in the top-right corner
3. **Click "Load unpacked"** button
4. **Select the BrowserAI folder**: Navigate to `/Users/aryansrivastava/Desktop/BrowserAI` and select it
5. The extension should now appear in your extensions list

## Step 2: Get Your Gemini API Key

1. Go to: https://makersuite.google.com/app/apikey
2. Sign in with your Google account
3. Click **"Create API Key"**
4. Copy the API key (it will look like: `AIzaSy...`)

3. **Click the "Exply's Explanation" button** that appears
4. Wait for the explanation to load

## Troubleshooting

**Extension not loading?**
- Make sure Developer Mode is enabled
- Check the browser console (`F12`) for any errors

**Button doesn't appear when highlighting text?**
- Make sure you've selected at least 8 characters
- Try refreshing the page and selecting again

**Getting API errors?**
- Verify your API key is correct in the options page
- Check that you're signed in to Google AI Studio
- Make sure you have API quota available
